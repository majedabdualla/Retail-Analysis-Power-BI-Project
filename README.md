# NexaRetail — Retail Analytics Power BI Project

A product-oriented **Retail Analytics Dashboard** built in **Power BI**, analyzing sales performance, profitability, geographic distribution, customer behavior, and operational insights.

---

## 📊 Project Overview

**NexaRetail** is a global retail company selling **consumer electronics and general merchandise** across multiple countries through 67 stores.

This project analyzes **transactional sales data from 2016 to 2021** using **Power BI** to uncover:

- Product performance insights
- Store efficiency
- Customer purchasing trends
- Geographic sales distribution
- Business bottlenecks impacting profitability

The goal is to transform raw transactional data into **interactive dashboards that support data-driven business decisions.**

---

##  Tools & Technologies

- **Power BI**
- **Power Query** (Data Cleaning & Transformation)
- **DAX** (Measures & KPIs)
- **Data Modeling**
- **Interactive Data Visualization**

---

## 📁 Dataset

The dataset includes transactional retail data containing:

- Orders
- Products
- Categories & Subcategories
- Stores
- Customers
- Geographic information
- Revenue, Cost, and Profit metrics

Time Range: **2016 — 2021**
